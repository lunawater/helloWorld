How to start sewing.

Its important that you get the thread throught the eye, and if the string is split at the end making that hard, just apply a little bit of water to the end of it.Remember, even if its alittle frustrtaing, the end result is worth it!

Fabic tips.

Try to start off with fabric that has a little more give, that way if the fabric patterens are cut wrong, mistakes arent as noticable.

good stiches to use.

    There are 3 types of stiches,
  1. zig zag stitch.
  2. running stitch.
  3. tack stitch.

The running stitch is the best to use if youre first tarting off, it allows for the user to sew more rounded patterns.
